# Halpt 🌟

**Rent + Learn from Your Local Community**


## 🎯 Overview

Halpt is a hybrid marketplace that combines item rentals with skill exchange. It connects local community members who want to rent items with those who can teach skills related to those items, creating a unique peer-to-peer learning and sharing ecosystem.

### ✨ Key Features
- **🔁 Item Rentals** - Rent everyday items from neighbors
- **🎓 Skill Exchange** - Learn from verified local mentors
- **🤝 Community Building** - Create micro-communities of learners and teachers
- **⭐ Reputation System** - Build credibility with badges and reviews
- **📍 Local Focus** - Connect with people in your neighborhood
